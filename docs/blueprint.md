# **App Name**: LearnAI

## Core Features:

- AI Tutor: Allows users to type or speak questions and receive detailed answers, examples, and action steps powered by AI. The AI will be equipped with a tool to reason about its confidence levels for a given answer.
- Personalized Learning Paths: AI tracks user progress and suggests lessons, quizzes, and resources based on skill level.
- Document & Code Analysis: Users can upload documents, code snippets, or images for AI analysis and receive summaries, corrections, or suggestions.
- Professional Dashboard: Allows users to track progress, achievements, and skills learned and shows completion certificates or badges.
- User Authentication: Implements user login with Google, email, or social using Firebase Auth.
- Data Storage: Stores user progress, queries, and results using Firestore.
- Notifications: Utilize Firebase Cloud Messaging to notify users when AI responses are ready.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a professional and trustworthy feel.
- Background color: Very light grey (#F0F2F5), a desaturated variant of the primary color, to provide a neutral backdrop.
- Accent color: Teal (#009688), an analogous color to the primary that will pop nicely on the UI.
- Body font: 'Inter' (sans-serif) for body text, to ensure legibility and a clean modern look.
- Headline font: 'Space Grotesk' (sans-serif) for headlines, paired with 'Inter', offering a tech-focused yet accessible feel.
- Use consistent, professional icons for easy navigation.
- Smooth transitions and subtle animations for an engaging user experience.
